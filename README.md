# 🧮 Calculadora Simples

Uma calculadora simples desenvolvida em Python com tratamento robusto de erros. Realiza operações básicas: soma, subtração, multiplicação e divisão.

## ⭐ Características

- ✅ Operações básicas: soma, subtração, multiplicação e divisão
- ✅ Validação completa de entrada do usuário
- ✅ Proteção contra divisão por zero
- ✅ Mensagens de erro claras e amigáveis
- ✅ Tratamento de exceções com try/except
- ✅ Código bem estruturado e documentado

## 📦 Pré-requisitos

- Python 3.6 ou superior
- Nenhuma biblioteca externa necessária (usa apenas stdlib)

## 🔧 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/Joaomb15/aula-3-projeto-final-reposit-rio-colaborativo-giteam.git
cd aula-3-projeto-final-reposit-rio-colaborativo-giteam
```

2. Execute o programa:
```bash
python calculadora.py
```

## 💻 Como Usar

Execute o programa e escolha qual operação deseja realizar:

```bash
$ python calculadora.py

Digite o primeiro valor: 10
Digite o segundo valor: 5
A soma entre 10 e 5 é: 15.0
```

### Operações Disponíveis

- **Soma**: Adiciona dois números
- **Subtração**: Subtrai o segundo número do primeiro
- **Multiplicação**: Multiplica dois números
- **Divisão**: Divide o primeiro número pelo segundo (com proteção contra zero)

## 🛡️ Tratamento de Erros

O programa possui validação completa:

- ❌ **Entrada inválida**: Se você digitar texto em vez de número
- ❌ **Divisão por zero**: Não permite dividir por zero
- ❌ **Exceções inesperadas**: Captura qualquer erro não previsto

Exemplo:
```bash
Digite o dividendo: 10
Digite o divisor: 0
Erro: Não é possível dividir por zero.
```

## 📸 Exemplos

### Multiplicação
```bash
Digite o primeiro fator: 7
Digite o segundo fator: 8
O produto entre 7.0 e 8.0 é 56.0
```
